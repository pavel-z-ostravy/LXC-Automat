// how it works

Up and running in minutes

01
Run the install script
One curl command clones the repo, installs dependencies, and starts the service on port 8091.
02
Open the web wizard
Visit http://<your-ip>:8091/setup — an 8-step wizard guides you through every setting. No SSH, no config files.
03
Dashboard starts automatically
After the wizard saves config.json, the app restarts as a full dashboard. Systemd keeps it running.
04
Everything is configurable live
Modules, WoL devices, services, credentials — all editable from the Settings panel without re-running the wizard.
// quick install
curl -sSL https://raw.githubusercontent.com/\
    pavel-z-ostravy/LXC-Automat/main/install.sh \
  | sudo bash

# Then open:
http://<your-server-ip>:8091/setup

// security

Security model

⚠️ Designed for trusted local networks only

Do not expose this dashboard directly to the internet. It has no HTTPS, no brute-force protection, and no rate limiting. For remote access, use a reverse proxy (Nginx / Caddy) with HTTPS, or a VPN / Cloudflare Tunnel.

🔑
Credentials never in plaintext
Passwords stored as SHA-256 hash only. config.json and keys/ are gitignored — credentials never reach the repository.
📱
Optional TOTP 2FA
TOTP secret stored in config.json with 600 permissions, never logged. Zero overhead when 2FA is disabled.
🔒
SSH keys protected
Generated ed25519 keypairs have 600 permissions. Key paths validated against the keys directory to prevent path traversal.
🧹
Input validation throughout
Shell commands use shlex.quote(). IDs, IPs, hostnames validated with strict regex. No shell=True on user input.
